# llmstxtscan.org

Public information site for the **llms.txt crawler bot** — a non-commercial
research crawler that checks websites for a small set of standard, publicly
served "well-known" files (`/llms.txt`, `/llms-full.txt`, `/AGENTS.md`,
`/.well-known/security.txt`, `/robots.txt`).

The site is a single static page (`index.html`) served via **GitHub Pages**
on the custom domain **https://llmstxtscan.org** (see `CNAME`). Its purpose is
transparency: it lets any site operator identify the crawler from its
User-Agent and request exclusion.

- **User-Agent:** `llms.txt crawler bot (+mailto:opt-out@llmstxtscan.org)`
- **Opt-out:** opt-out@llmstxtscan.org
- **Contact:** contact@llmstxtscan.org

Research conducted in association with the
[Cloud Security Alliance](https://cloudsecurityalliance.org).
